# Builder Feedback Sprint

Goal: get 10 real Mac/Codex users through the first install path and fix the first blocker they hit.

Feedback link:

```text
https://github.com/dicnunz/codex-relay/issues/new?template=install-feedback.yml
```

## What To Ask

Give it 15 minutes:

1. Create a dedicated Telegram bot with `@BotFather`.
2. Run:

   ```bash
   git clone https://github.com/dicnunz/codex-relay.git
   cd codex-relay
   ./scripts/install.sh
   ```

3. DM the bot:

   ```text
   /alive
   /health
   /policy
   /screenshot
   ```

4. Try one real local task in a safe folder.
5. Open an install feedback issue with the first confusing, broken, or slow part.

Do not paste bot tokens, `.env`, private screenshots, personal files, raw Codex transcripts, or unredacted logs.
